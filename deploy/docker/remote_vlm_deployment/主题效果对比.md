## 🎯 使用场景推荐

### 推荐配置矩阵

| 使用场景 | 推荐主题 | 理由 |
|---------|---------|------|
| **政府演示** | 🟣 浅紫色 | 正式、专业、中性 |
| **公安系统** | 🟣 浅紫色 | 庄重、权威、无品牌色 |
| **企业内部** | 🔷 科技黑 | 现代、科技、护眼 |
| **技术展会** | 🔷 科技黑 | 吸睛、酷炫、科技感 |
| **开发调试** | 🔷 科技黑 | 护眼、长时间使用 |
| **合作演示** | 🟢 原始主题 | 保留原始风格 |

## 🔄 快速切换命令

```bash
# 切换到浅紫色（推荐）
bash set_theme.sh purple

# 切换到科技黑
bash set_theme.sh dark

# 恢复原始主题
bash set_theme.sh original
```

## 🎨 颜色心理学

### 浅紫色（Purple）
- 象征：高贵、智慧、创新
- 适合：政府、科技、创意行业
- 情感：专业、可信、现代

### 科技黑+青蓝（Dark Cyan）
- 象征：科技、未来、精确
- 适合：科技公司、研发团队
- 情感：酷炫、专注、前沿

### NVIDIA 绿（Green）
- 象征：能量、成长、科技
- 适合：NVIDIA 相关场景
- 情感：活力、专业、创新
