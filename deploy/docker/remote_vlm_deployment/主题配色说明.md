## 🚀 快速切换主题

### 方法 A：一键切换（推荐）

```bash
cd /Users/cauchy/gitrepo/video-search-and-summarization/deploy/docker/remote_vlm_deployment

# 切换到浅紫色主题
bash set_theme.sh purple

# 切换到科技黑主题
bash set_theme.sh dark

# 恢复原始主题
bash set_theme.sh original
```

### 方法 B：手动执行

```bash
# 1. 进入容器
docker exec -it via-server bash

# 2. 执行主题切换
python3 /opt/nvidia/change_theme.py purple

# 3. 退出并重启
exit
docker-compose restart via-server
```

## ⚠️ 注意事项

1. **主题持久性**
   - 主题配置在容器重建后会丢失
   - 需要重新执行 `bash set_theme.sh <主题名>`

2. **浏览器缓存**
   - 切换主题后请强制刷新：`Ctrl + F5` (Windows/Linux) 或 `Cmd + Shift + R` (Mac)

3. **最佳实践**
   - 首次部署后立即设置主题
   - 建议将主题切换命令加入部署文档

## 🎨 自定义主题

如需自定义配色，编辑 `change_theme.py` 中的 `THEMES` 字典即可添加新主题。

**示例：添加粉色主题**
```python
"pink": {
    "name": "粉色主题",
    "button_primary_background_fill": "#ec4899",
    "button_primary_border_color": "#ec4899",
    "logo_fill": "#EC4899",
    "color_accent": "#ec4899",
    # ... 其他配色
}
```

## 📞 问题反馈

如遇到主题切换问题：
1. 检查容器是否运行：`docker ps | grep via-server`
2. 查看日志：`docker logs via-server`
3. 强制刷新浏览器：`Ctrl + F5`
4. 重新应用主题：`bash set_theme.sh purple`
